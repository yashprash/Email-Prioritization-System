Answering A Few FAQ


What is Email Prioritization System?

The email prioritization system can be integrated with a userâ€™s existing mailbox. The front-end user interface of the system has been designed using the Django framework. The classification of emails into the category folders is performed using the multiclass classification output vector generated by a recurrent neural network model, trained using a dataset of 10000 emails of the open source Enron email dataset.

What are the advantages of this System?

The email prioritization system takes 3 categories of interest (out of 6 available options) as input from the user, and classifies all incoming emails into one of the three categories of interest, or into a separate folder if the email does not match any of the categories selected by the user

Why use this system?

This system is advantageous as it provides the user with a personalized way to access and view his/her emails, according to what the user considers to be important, and selects as his/her top three priority categories.

Technologies Used

Natural Language Processing, Supervised Learning, Recurrent Neural Network, Front End using Django Framework, The Model hosted on AWS with Flask Server
